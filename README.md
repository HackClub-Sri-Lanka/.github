# 🇱🇰 Hack Club Sri Lanka

Welcome to the official open source organization of **Hack Club Sri Lanka**, Sri Lanka's very first club registered under @hackclub
We are a student-led community empowering teenagers across Sri Lanka to build, learn, and ship real software together.

We are building a generation of young developers, designers, hackers, makers, and founders through open source collaboration, hackathons, workshops, and community-driven projects.

> Registered under [Hack Club](https://hackclub.com/), a global nonprofit network of coding clubs and teen hackers.

---

## 🚀 What We Do

Hack Club Sri Lanka is a place where students

* Build real-world software projects
* Contribute to open source
* Learn modern technologies together
* Participate in hackathons and workshops
* Collaborate with developers from all around Sri Lanka
* Create impactful solutions for communities

This GitHub organization is where our community publishes and maintains open source projects created by members of Hack Club Sri Lanka.

---

## 🤝 Contributing

We welcome contributions from students, beginners, and experienced developers alike.

Before contributing, please make sure to,

### ✅ Contribution Rules

* Be respectful and supportive to everyone
* Write clean and understandable code
* Use meaningful commit messages
* Test your code before submitting
* Follow the project's coding style
* Avoid uploading unnecessary files
* Do not submit AI-generated spam pull requests

---

## 📌 Pull Request Guidelines

When creating a Pull Request:

1. Fork the repository
2. Create a new branch for your feature/fix
3. Make your changes
4. Commit your changes clearly
5. Open a Pull Request with a proper description

### PR Title Example

```
feat: added dark mode support
fix: resolved login issue
docs: updated setup instructions
```
